# FICO&reg; Xpress third-party interface examples

Integration examples for using [FICO&reg; Xpress Solver](https://www.fico.com/en/products/fico-xpress-optimization) with third-party optimization modeling libraries.

## Contents

Library | Language | Description | Blog
--------|----------|-------------|------
[OR-Tools](ortools/) | C++, Python | Google's Operations Research tools with Linear Solver and MathOpt interfaces | TBD

More integrations coming soon: PyOptInterface, Pyomo, PuLP, linopy, JuMP, MiniZinc, CVXPY.

## Requirements

- [FICO&reg; Xpress](https://www.fico.com/en/products/fico-xpress-optimization) (Community Edition or licensed)
- Language-specific requirements vary by example (see individual READMEs)
